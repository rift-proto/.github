= Rift

Rift Realtime Protocol (Rift/1) — a modern real-time bidirectional messaging protocol designed to replace raw WebSocket + event-string models with stronger semantics.

== Repositories

[cols="1,2"]
|===
| Repository | Description

| https://github.com/rift-proto/rift-rust[rift-rust]
| Server-side implementation (`rifts` crate). Supports standalone WebSocket and framework adapters (Axum, Actix Web, Warp, Ntex).

| https://github.com/rift-proto/rift-ts[rift-ts]
| TypeScript client SDK for browser and Node.js.
|===

== Protocol Highlights

* Typed frame envelope with priority and flags
* CBOR + JSON codec negotiation
* Eight message classes (event, command, reply, state, etc.)
* Topic-based pub/sub with retention and ordering
* Session management with auth and resume
* Acknowledgement system (9 ack types)
* Backpressure control and token-bucket rate limiting
